## Slots

- **base**: The base slot of the motionBlurLoader, it wraps the circles and the label.
- **wrapper**: The wrapper of the circles.
- **circle1**: The first circle of the motionBlurLoader.
- **circle2**: The second circle of the motionBlurLoader.
- **circle3**: The third circle of the motionBlurLoader.
- **label**: The label content.

## API

### Circular Progress Props

| Attribute  | Type                                                                        | Description                                             | Default   |
| ---------- | --------------------------------------------------------------------------- | ------------------------------------------------------- | --------- |
| label      | `string`                                                                    | The content to display as the label.                    | -         |
| size       | `xs` \| `sm` \| `md` \| `lg`                                                        | The size of the motionBlurLoader circles.                        | `md`      |
| color      | `default` \| `primary` \| `secondary` \| `success` \| `warning` \| `danger` | The color of the motionBlurLoader circles.                       | `primary` |
| labelColor | `default` \| `primary` \| `secondary` \| `success` \| `warning` \| `danger` | The color of the label.                                 | `default` |
| classNames | `Record<"base"｜"wrapper"｜"circle1"｜"circle2"｜"circle3"｜"label", string>`          | Allows to set custom class names for the motionBlurLoader slots. | -         |
